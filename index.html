Items are the data type used in the visualization above with each data point being a node in the network. In this case, the mark is a connnection (link). The visual is used to visually organise information where nodes are then asssoicated with one another which ultimately connect to the central concept (THEME). As for channels, color is utilized to color code which node is the central concept, and also may have correlation with the other nodes, but it is not clear, as nodes with the same name are provided with a different color.
